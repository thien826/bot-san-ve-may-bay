Hướng Dẫn Cấu Hình
1. Telegram Bot: Nhắn /newbot cho @BotFather → nhận TOKEN
2. Chat ID: Dùng @ten_kenh (kênh public) hoặc -100xxxxxxx (nhóm)
3. Biến môi trường cần thêm vào Render.com:
TELEGRAM_TOKEN=your_token
TELEGRAM_CHAT_ID=@your_channel
AMADEUS_API_KEY=your_key      # tuỳ chọn
AMADEUS_API_SECRET=your_secret
APP_URL=https://your-app.onrender.com
4. Amadeus API (miễn phí): developers.amadeus.com → Sign Up → My Apps
